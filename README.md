Nombre de la aplicación: 


GameVault

Descripción breve:


GameVault es una aplicación móvil desarrollada con React Native que simula una tienda de videojuegos. tiene un catálogo de juegos, ver detalles completos de cada título, descripción, edad para jugar, genero y precio, agregar nuevos juegos mediante un formulario y consultar noticias del mundo juegos con actualización automática.

Instrucciones de instalación y ejecución:


clonar repositorio: https://github.com/Vivi2210/GameVault


instalar dependencia: npm install


npx react-native run-android

Tecnologías utilizadas:


React Native

React Navigation (Stack y Tab Navigation)

Hooks de React (useState, useEffect)

StyleSheet para estilos

SafeAreaView y KeyboardAvoidingView
